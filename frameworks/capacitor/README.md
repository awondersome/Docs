capacitor 可以嵌入到现有的前端项目，需要更多的参与native的配置，由native控制，更像是native的一部分，可以看成是一个配置文件
cordova 是前端代码嵌入到cordova项目

npm install --save @capacitor/core @capacitor/cli

npx cap -V 查看版本

npx cap init 输入包名和id

npx cap add


npx cap copy

npx cap open
